// GOOD QUESTIONS

Frequently asked

Is it safe to generate a password here?

Yes — completely. The password is created in your browser using secure randomness, and it’s never sent to us or anywhere else. Nothing is stored.

What makes a strong password?

Length and unpredictability. A long random password (16+ characters) or a passphrase of several random words is far stronger than a short complicated one — and the golden rule is a different password for every account.

Random password or memorable passphrase?

A random password is strongest and perfect for a password manager to remember for you. A passphrase (like ‘Otter-Lantern-River-42’) is easier to type and recall — ideal for the few passwords you need to know by heart, such as your password manager itself.

How do I keep track of all my passwords?

A password manager — it remembers them all behind one strong password and fills them in for you. We set them up for homes and businesses and show you how. Ask us.

Should I check if my old passwords are safe?

Good idea — try our password breach checker to see if any have appeared in a data breach, and change any that have.
