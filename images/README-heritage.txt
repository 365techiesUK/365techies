HERITAGE PHOTOS — "Our Dorset computing heritage" gallery on /about/
====================================================================
Real, owner-supplied photos. Processed to <=1400px / optimised JPEG. The gallery
(build_pages.py heritage_gallery() + HERITAGE_IMAGES) renders only files that exist.

Source of the shop photos: owner's Google Business Profile. Method: owner shares each photo's
maps.app.goo.gl link; the redirect URL embeds a direct lh3.googleusercontent.com image URL that
can be downloaded (append =w2000). To add more, drop a JPEG here + add a line to HERITAGE_IMAGES,
then rebuild (python build_blog.py). Orient upright first.
