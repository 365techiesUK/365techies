// GOOD QUESTIONS

Frequently asked

Which phones and tablets do you support?

We support Android phones and tablets, Samsung devices and Windows tablets. Please note we don’t currently offer remote support for Apple iPhones or iPads.

Can you set up email on my phone?

Yes — we set up and fix email on your Android phone or tablet, including Microsoft 365 and business email.

Can you back up my photos and contacts?

Yes — we set up reliable photo, contact and file backup so nothing’s ever lost.

Can you set up my new phone?

Yes — we move everything across to your new Android phone or tablet and get it working.
